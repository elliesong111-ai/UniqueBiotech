# 🥛 为什么美国的奶比中国的奶好喝？| 深度科普

一个交互式的科普网站，从六个维度深度解析美国牛奶的品质优势。

## 📋 项目简介

本项目通过交互式网页的形式，从以下六个维度全面解析美国牛奶的优势：

1. **📜 政策与循环经济** - 可持续发展和环保政策
2. **🏡 养殖技术** - 现代化养殖管理
3. **🌾 饲料与营养** - 科学配方的饲料体系
4. **🏭 加工技术** - 先进的加工工艺
5. **💚 牛的健康** - 完善的健康管理体系
6. **⚙️ 系统整合** - 全产业链优化

## 🚀 功能特点

- ✨ **交互式界面** - 点击卡片即可查看详细内容
- 📊 **数据可视化** - 直观的对比图表和统计信息
- 📱 **响应式设计** - 完美适配桌面和移动设备
- 🎨 **现代化UI** - 美观的界面设计和流畅的动画效果
- ♿ **无障碍支持** - 支持键盘导航和屏幕阅读器

## 🛠️ 技术栈

- **HTML5** - 语义化标记
- **CSS3** - 现代化样式和动画
- **JavaScript (ES6+)** - 交互功能实现
- **响应式设计** - 移动优先的设计理念

## 🚀 快速开始

### 本地运行

1. 克隆或下载此仓库
2. 直接在浏览器中打开 `index.html` 文件
3. 或者使用本地服务器：

```bash
# 使用Python
python -m http.server 8000

# 使用Node.js (需要安装http-server)
npx http-server

# 使用PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

### 部署到GitHub Pages

1. 在GitHub上创建新仓库
2. 将文件推送到仓库
3. 在仓库设置中启用GitHub Pages
4. 选择主分支（main/master）作为源
5. 访问 `https://你的用户名.github.io/仓库名`

## 🎨 自定义

### 修改颜色主题

在 `styles.css` 文件的 `:root` 选择器中修改CSS变量：

```css
:root {
  --primary-color: #4a90e2;
  --secondary-color: #50c878;
  /* ... 其他颜色变量 */
}
```

### 添加新内容

1. 在 `index.html` 中添加新的维度内容区域
2. 在 `script.js` 的 `dimensionMap` 中添加映射关系
3. 在 `dimensions-grid` 中添加新的卡片
