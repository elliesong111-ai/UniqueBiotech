# 测试指南

## 🧪 功能测试清单

### 基本功能测试

- [x] **页面加载**
  - 打开 `index.html`，检查页面是否正常加载
  - 检查加载动画是否显示并消失
  - 验证所有样式是否正确应用

- [x] **导航功能**
  - 点击导航栏的"简介"和"六大维度"链接，检查是否平滑滚动
  - 验证导航栏在滚动时是否保持固定

- [x] **维度卡片交互**
  - 点击每个维度卡片（6个卡片）
  - 验证点击后是否正确显示对应内容
  - 检查卡片悬停效果是否正常

- [x] **内容显示**
  - 验证每个维度的详细内容是否正确显示
  - 检查美国与中国的对比信息是否完整
  - 验证统计图表是否正确显示

- [x] **返回功能**
  - 点击"← 返回"按钮
  - 验证是否返回到维度选择区域
  - 检查默认内容是否显示

- [x] **键盘导航**
  - 按 ESC 键，验证是否返回主菜单
  - 检查键盘导航是否流畅

### 响应式测试

- [x] **桌面端（> 768px）**
  - 检查布局是否正常
  - 验证网格布局是否正确显示
  - 检查所有元素是否对齐

- [x] **平板端（768px - 1024px）**
  - 检查响应式布局是否适配
  - 验证卡片网格是否调整

- [x] **移动端（< 768px）**
  - 检查移动端布局
  - 验证导航栏是否适配
  - 检查触摸交互是否流畅
  - 验证文字大小是否可读

### 浏览器兼容性测试

建议在以下浏览器中测试：

- [x] **Chrome/Edge** (最新版本)
- [x] **Firefox** (最新版本)
- [x] **Safari** (最新版本)
- [x] **移动浏览器** (iOS Safari, Chrome Mobile)

### 性能测试

- [x] **加载速度**
  - 检查页面首次加载时间
  - 验证字体加载是否影响体验
  - 检查动画是否流畅

- [x] **交互响应**
  - 验证点击响应是否及时
  - 检查滚动是否流畅
  - 验证动画是否流畅

### 内容检查

- [x] **文本内容**
  - 检查所有文本是否正确显示
  - 验证中文字体是否正确加载
  - 检查是否有错别字

- [x] **数据准确性**
  - 验证统计数据是否正确
  - 检查对比信息是否完整
  - 验证引用来源是否标注

## 🔧 测试方法

### 本地测试

1. **直接打开**
   ```bash
   # 在文件管理器中双击 index.html
   # 或在浏览器中打开
   ```

2. **使用本地服务器**
   ```bash
   # Python
   python -m http.server 8000
   
   # Node.js
   npx http-server
   
   # PHP
   php -S localhost:8000
   ```

3. **访问**
   - 打开浏览器访问 `http://localhost:8000`

### 移动端测试

1. **使用浏览器开发者工具**
   - 按 F12 打开开发者工具
   - 点击设备工具栏图标
   - 选择不同设备进行测试

2. **真实设备测试**
   - 将文件上传到服务器
   - 在真实移动设备上访问
